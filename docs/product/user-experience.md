# User Experience & Interface

## Screens

### 1. Home / Landing
- Play Now (quick match)
- Create Private Game (share link)
- Join Game (enter code)
- How to Play (rules overview)

### 2. Lobby / Waiting Room
- Player list with ready status
- Game settings (target score, variant)
- Start game button (host only)
- Share invite link/code

### 3. Game Board
- Central play area showing the domino chain
- Player's hand at the bottom
- Opponent hands shown as face-down tile counts
- Boneyard count indicator
- Current turn indicator
- Score display
- Timer per turn (optional)

### 4. End of Round / Game Over
- Round scores breakdown
- Running total
- Play Again / Return to Lobby

## Interaction Design

- **Drag & drop** tiles to play (desktop)
- **Tap to select, tap to place** (mobile)
- Valid placements highlighted when a tile is selected
- Smooth animations for tile placement
- Subtle haptic feedback on mobile (if supported)

## Visual Design

- Clean, minimal aesthetic
- High contrast tiles for readability
- Dark mode support
- Responsive layout — works on any screen size
