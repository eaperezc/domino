# Game Rules & Mechanics

## Standard Rules: Draw Dominoes (Double-Six)

### Setup
- 28-tile double-six set (each tile has two ends, values 0–6)
- 2 players: 7 tiles each
- 3 players: 7 tiles each
- 4 players: 7 tiles each (0 in boneyard)
- Remaining tiles form the **boneyard** (draw pile)

### Gameplay
1. Player with the highest double starts (or random if no doubles)
2. Players take turns clockwise
3. On your turn, play a tile matching an open end of the chain
4. If you cannot play, draw from the boneyard until you can (or boneyard is empty)
5. If you cannot play and boneyard is empty, pass your turn

### Winning
- First player to play all their tiles wins the round
- If the game is **blocked** (no one can play), the player with the lowest pip count wins
- Winner scores the total pip count of all opponents' remaining tiles

### Scoring
- Games can be played to a target score (e.g., 100, 150, or 200 points)
- Or single-round mode for quick games

## Future Variants

- **Block Dominoes**: No boneyard draws — if you can't play, you pass
- **All Fives**: Score points when open ends sum to a multiple of 5
- **Mexican Train**: Extended set with train mechanics
