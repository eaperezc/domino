# Player Profiles & Progression

## MVP (Phase 1)
- Anonymous play with display name (stored in local storage)
- No account required to start playing

## Phase 2 — Accounts
- Optional sign-up (email / OAuth)
- Persistent stats: games played, win rate, avg score
- Match history
- ELO rating

## Phase 3 — Progression
- Achievements (e.g., "Win 10 games", "Play a double-six opener")
- Leaderboards (global, weekly, friends)
- Customizable tile themes / table backgrounds
